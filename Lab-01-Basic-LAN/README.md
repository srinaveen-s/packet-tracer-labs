# Lab 01 - Basic LAN

## 🎯 Objective

Connect two computers through a switch and verify communication using Ping.

---

## 🖥 Devices Used

- Cisco 2960 Switch
- PC0
- PC1

---

## 🌐 IP Addressing

PC0

192.168.1.10

PC1

192.168.1.20

Subnet Mask

255.255.255.0

---

## 🧪 Verification

Ping from PC0 to PC1.

---

## ✅ Expected Result

Ping should be successful.

---

## 📚 Skills Learned

- Basic LAN
- IP Addressing
- Connectivity Testing
