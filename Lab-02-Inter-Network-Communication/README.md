# Lab 02 - Inter-Network Communication Using a Router

## 🎯 Objective

To connect two different networks using a Cisco router and verify communication between devices.

---

## 🛠 Devices Used

- 1 Cisco 2911 Router
- 2 Cisco 2960 Switches
- 2 PCs

---

## 🌐 Network Details

### Network 1

Admin-PC

IP Address: 192.168.1.10

Subnet Mask: 255.255.255.0

Default Gateway: 192.168.1.1

---

### Network 2

Client-PC

IP Address: 192.168.2.10

Subnet Mask: 255.255.255.0

Default Gateway: 192.168.2.1

---

## ⚙ Router Configuration

Router Interface G0/0

IP Address: 192.168.1.1

Subnet Mask: 255.255.255.0

Router Interface G0/1

IP Address: 192.168.2.1

Subnet Mask: 255.255.255.0

---

## 🧪 Verification

A successful ping was performed from Admin-PC to Client-PC.

---
