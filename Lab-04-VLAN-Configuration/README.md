# Lab 04 - VLAN Configuration

## 🎯 Objective

Create multiple VLANs on a Cisco switch and assign different ports to different VLANs.

---

## 🛠 Devices Used

- Cisco 2960 Switch
- 4 PCs

---

## VLAN Details

VLAN 10 - HR

VLAN 20 - Finance

VLAN 30 - IT

---

## Verification

Devices within the same VLAN can communicate.

Devices in different VLANs cannot communicate without Inter-VLAN Routing.

---

## Learning Outcome

After completing this lab, I learned:

- VLAN Creation
- Port Assignment
- Broadcast Domain Separation
- Basic Switch Configuration
